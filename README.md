
# Supply Chain Demand Forecasting using Temporal Fusion Transformer (TFT)

## Overview
This project implements an AI-powered demand forecasting system for FMCG retail supply chains using:
- PyTorch Forecasting
- Temporal Fusion Transformer (TFT)
- SHAP Explainability
- FastAPI Deployment
- Docker
- Automated Training Pipeline

## Features
- Multi-horizon demand forecasting
- Promotion and holiday-aware forecasting
- Explainable AI using SHAP
- FastAPI REST API deployment
- Dockerized environment
- AWS deployment ready
